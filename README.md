# Visual-Computing

This repository contains a collection of small **Processing 4** projects that I worked on during my Master's studies. Each project is organized in its own folder.

## How to View

1. **Download the repository** as a ZIP file or clone it to your local machine.
2. **Open Processing 4**.
3. Navigate to the folder of the project you want to view and open it in Processing.

## Projects

Each project folder includes the source code and any assets needed to run the sketch. Simply open the `.pde` file in Processing 4 to run it.

Feel free to explore and experiment with the sketches!
